## Development

### For web apps

Npm

	$ npm install --save unnamed

Yarn

	$ yarn add unnamed

Usage with webpack:

	import 'unnamed'

### For landing pages

#### Clone & start dev server

	$ git clone https://github.com/smakosh/Unnamed-getting-started.git

	$ cd Unnamed-getting-started && yarn install && bower install

	$ gulp serve

#### To Build

	$ gulp build
